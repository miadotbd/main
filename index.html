না, বাংলাদেশ ডোমেইন অথরিটি সাবডোমেইন বিক্রি নিষিদ্ধ করেছে। আমরা সাবডোমেইন ফ্রি দিচ্ছি, তবে স্মার্ট প্রোফাইল কার্ড এবং ওয়েবসাইট সেবার মাধ্যমে ব্যবসা করছি।

সাবডোমেইনের জন্য DNS নিয়ন্ত্রণ থাকবে (A, CNAME, TXT রেকর্ড)। GitHub Pages হোস্টিংয়ে সম্পূর্ণ অ্যাকাউন্ট নিয়ন্ত্রণ থাকবে এবং WordPress হোস্টিংয়ে পূর্ণ অ্যাডমিন অ্যাক্সেস দেওয়া হবে।

স্মার্ট প্রোফাইল কার্ড হলো একটি আধুনিক পরিচয় কার্ড, যেখানে আপনার নাম, ওয়েবসাইট এবং ডিজিটাল তথ্য থাকবে। এটি আপনার ব্যক্তিগত ব্র্যান্ডকে অনলাইনে তুলে ধরে।

হ্যাঁ, ওয়েবসাইট 100% কাস্টমাইজযোগ্য। আপনি যেকোনো সময় কনটেন্ট আপডেট, পরিবর্তন বা ডিলিট করতে পারবেন।

আমরা তিনটি প্যাকেজ দিচ্ছি: ফ্রি সাবডোমেইন, লাইফটাইম প্যাকেজ (৳৩০০০ একবারেই), এবং বার্ষিক প্যাকেজ (৳১০০০ প্রতি বছর)। লাইফটাইম প্যাকেজে GitHub Pages হোস্টিং থাকবে এবং বার্ষিক প্যাকেজে WordPress হোস্টিং থাকবে।
